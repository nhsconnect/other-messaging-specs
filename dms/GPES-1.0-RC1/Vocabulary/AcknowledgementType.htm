Vocabulary for AcknowledgementType

Version: 4.0
Status: active
Date: 2007-01-18

Description

A code describing the action taken on the message by the receiving application and carried by the Acknowledgement.typeCode.

Values

Code Description Note
AA Application Acknowledgement Accept

Receiving application successfully processed message.
AE Application Acknowledgement Error

Receiving application found error in processing message. Additional error detail information is returned to the sender via the control act and optionally via a response payload.
AR Application Acknowledgement Reject

Receiving application failed to process message for reason unrelated to content or format. Error is identified in the application acknowledgement. Original message sender must decide on whether to automatically send message again.
