Vocabulary for HL7StandardVersionCode

Version: 30.0
Status: active
Date: 2009-03-09

Description

Describes the version of HL7 V3 used to format the messages in a specific version of the MIM.

The versionCode attribute of the message interaction shall be assigned the code from the following set that corresponds to the MIM version in which this code system version appears.
