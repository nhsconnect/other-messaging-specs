Overview: General Practice Extraction Service Domain Message Specification

The General Practice Extraction Service (GPES) Domain Message Specification supports the following care communications:

 

  • GPES-Q-E-NT Notification message.
  • GPES-E-Q-NT Notification message.
  • GPES-Q-E-RTP Run Time Parameter Request message.
  • GPES-E-Q-RTPR Run Time Parameter Response message.

Also included in this Domain Message Specification are artefacts required to instantiate the message specifications.

These include:

  • TMS Infrastructure Specification
  • CMETs Specification for CMETs used with TMS Infrastructure
  • Data Types Specification
  • Vocabulary Specification for Vocabularies used by this domain
