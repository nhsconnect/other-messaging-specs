Vocabulary for Prescription Charge Exemption

 

Description:

Default Scheme:   NHS Data Dictionary  TBA

Assigned OID:   2.16.840.1.113883.2.1.3.2.4.16.33

Describes possible reasons for someone not having to pay prescription charges.

 

The age exemption category (0002 and 0004) will be calculated by the system based upon the patient’s date of birth contained within the prescription message. Message code 0001 is dependant upon the value entered for me medication payment charge codes (see section 14). 

 

This will generally be present in an Observation act where the code attribute has an id value of “EX” from the Prescription Annotation vocabulary.
