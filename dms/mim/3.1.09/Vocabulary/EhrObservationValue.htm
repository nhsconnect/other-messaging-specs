Vocabulary for EhrObservationValue

  

Description:

The ObservationStatement value may contain a code that provides a value associated with the accompanying code.

 

Three specific purposes are supported:

  • Episode;
  • Significance;
  • Use of drug code with a code indicating allergy or adverse reaction to a drug.

 

 

1          Episode codes

Indication of whether this is the first occurrence of a disorder or problem or whether it is new episode or a continuations of a current episode.
