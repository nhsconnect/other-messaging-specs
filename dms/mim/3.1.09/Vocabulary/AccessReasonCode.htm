Vocabulary for AccessReasonCode

 

 

Description

Assigned OID:         2.16.840.1.113883.2.1.3.2.4.17.7

Three types of Legitimate Relationship may be “manually” created by a user. These are contained within the LRType vocabulary. They are:

·       Self-claimed

·       Colleague granted

·       Expressed patient consent to access.

 

If any of the above LRType are selected then the user must select a AccessReasonCode from the list below. Please note that the list is constrained depending upon the LRType code.
