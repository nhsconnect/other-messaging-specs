Vocabulary for SupplyRequestSnCT

 

Description:

Assigned OID:  the OID for the UK subset of SnomedCT, q.v.

Codes taken from the UK subset of SnomedCT representing a pharmaceutical product supply request (e.g. NHS prescription, hospital supply), and its current status (e.g. requested, accepted).

 

Legend:

A code of TBA indicates that the SnCT code is not yet available.

+ after a code indicates that descendants of the code shall be used.
