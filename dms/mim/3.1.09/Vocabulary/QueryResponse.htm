Vocabulary for QueryResponse

  

Description

A code classifying the general nature of a response to a query. This is a UK localised version of the HL7 vocabulary.

 

 

Values:

Version:

04

Date:

30th Nov 2004

 

 

Code

Description

Note

ID

Issue detected

An issue has been detected. This could be either an application error, an issue relating to the query specification or an issue qualifying a response. A response may be present.  Note that issues can be classified as error, warning, or information.

NF

No data found

Query specification valid but no data found. Not currently used within NPfIT.

OK

Data found

Query response data found matching the query specification. No issues highlighted.
