Vocabulary for MedicationAdministrationTypeSnCT

 

Description:

Assigned OID:  the OID for the UK subset of SnomedCT, q.v.

Codes taken from the UK subset of SnomedCT representing that a medication should be administered to a patient.  If required, it may indicate the routes (e.g. intravenous) and/or the site(s) of administration (e.g. left arm) to be used. 

 

Legend:

A code of TBA indicates that the SnCT code is not yet available.

+ after a code indicates that a descendant of the code shall be used.
