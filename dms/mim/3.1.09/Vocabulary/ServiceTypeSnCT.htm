Vocabulary for ServiceTypeSnCT

  

Description

Assigned OID: as per Snomed CT, qv the link below.

Any code from the SnomedCT UK subset that describes the service type that provides patient care, either directly or indirectly.  A code from SNOMED CT (UK) may be accompanied by additional SnomedCT(UK) qualifying codes. 

 

Codes are required for NHS Direct, and for the varieties of complementary medicine (accupuncture, homeopathy, etc)
