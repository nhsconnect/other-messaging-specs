Vocabulary for DetectedIssueQualifier

 

Description

Assigned OID:           2.16.840.1.113883.2.1.3.2.4.17.104

 

Qualifiers classifying the severity of the issue or error carried in the Detected Issue Event present in the Trigger Event Control Act and the Query Acknowledgement Response.

 

This qualifier code system is combined with other code systems as specified by the Qualification of Detected Issue code system.

Description

The qualifier classifies the code as a warning.

For Information

The qualifier identifies an issue for information.
