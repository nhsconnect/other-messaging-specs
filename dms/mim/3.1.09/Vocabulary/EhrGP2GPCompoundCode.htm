Vocabulary for EhrGP2GPCompoundCode

 

Description:

The value set depends on the containment level indicated by the classCode of the CompoundStatement as noted in subsequent sections.

 

Four specific levels are supported:

  • Topic;
  • Category;
  • Cluster;
  • Battery.

 

 

1          Topic codes

This value set only applies to CompoundStatements at 'topic' level (classCode='TOPIC').

 

It is used to name distinct topics dealt with in a single encounter or composition.
