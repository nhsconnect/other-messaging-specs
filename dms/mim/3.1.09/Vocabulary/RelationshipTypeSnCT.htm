Vocabulary for RelationshipTypeSnCT

  

Description

Assigned OID:           the SnomedCT OID, qv the link below

Any code from the UK subset of SnomedCT that describes a relationship between the contents of one clinical statement and another, for example 'outcome of', 'reason for', 'is supporting evidence for'.

 

As of 6th December, there is no concept of the type mentioned below in SnomedCT (UK), although several relationship types are listed under concept code 408739003 "unapproved attribute (attribute)", and under concept code 106234000 "general adjectival modifier".
