Vocabulary for ObservationTypeSnCT

  

Description

Assigned OID:           the SnomedCT OID, qv the link below

Any code from the UK subset of SnomedCT that describes a kind of observation about a person, for example for a sign, symptom, diagnosis or test result.

 

A code from Snomed-CT (UK) may be accompanied by additional SnomedCT qualifying codes that are valid for the concept being qualified.
