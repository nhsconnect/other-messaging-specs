Examples Index for ETP
Medication Management v1.5

This section of the MiM contains ETP examples grouped by scenarios. For the respective examples and the associated schemas, a validation report is also provided.

The example messages contained within this document use a 12 digit Prescriber Id code within the payload. Please note that this is incorrect and should be a 6 digit General National Code for GPs.
