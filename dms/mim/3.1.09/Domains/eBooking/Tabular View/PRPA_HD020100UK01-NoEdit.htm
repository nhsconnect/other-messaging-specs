PRPA_RM020000UK01  
ConfirmedService

The Confirmed Service represents a confirmation of a request for service from a Service Requestor.

Used by:
[1..1] classCode (CS {CNE:PCPR} ){ Fixed="PCPR " }

Indicates that this is a patient care provision.

  •  Fixed value: "PCPR"
[1..1] moodCode (CS {CNE:PRMS} ){ Fixed="PRMS " }

Indicates that this is a promise to fulfill a service request.

  • Fixed value: "PRMS"
[1..1] id (II)

This identifier is unique within the eBooking Service. It identifies the Unique Booking Reference Number(UBRN) allocated to the service request. 

The Identifier External datatype flavour is used to carry this information as follows:

  • The root attribute shall contain an OID with the value 2.16.840.1.113883.2.1.3.2.4.1
  • The extension attribute shall contain the UBRN

 Example

<id root="2.16.840.1.113883.2.1.3.2.4.1" extension="1000000001"/>
