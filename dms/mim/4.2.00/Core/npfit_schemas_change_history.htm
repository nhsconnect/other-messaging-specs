1.0 C & M Development 27/02/06 First version

Changes to NPfIT and NPfIT versions of HL7 Schemas for MIM 5.0

1.1        Changes to datatypes.xsd

  • Created a new ED flavour with xhtmlNPfITPresentationText as the content.
  • Imported the NarrativeBlock CDA mark-up schema for Lab domain use.

1.2        Changes to voc.xsd

  • Added "INT" to the ActMoodIntent_X simple type.

1.3        Changes to xhtmlNPfITPresentationText.xsd.

  • Element declaration for "html" replaced with a group.
  • Elements "head", "body", "br" replaced with type references.
  • Commented "h1" element deceleration removed.
  • Element's "p","ol","ul","li" were amended with missing attributes "id" and "class".
  • Element "table" content for "tbody" and "tr" changed from sequence to choice. This is to be inline with xhtml requirements.
  • Element "tr" content for sequence of "th" and "td" changed to choice. This is for meeting the same guidelines as above.
  • Commented section within element "th" was removed.

1.4        Changes to xhtmlNPfIT.xsd

  • Changed targetNamespace declaration from "http://www.w3.org/1999/xhtml" to "http://www.w3.org/1999/xhtml-NPfIT". This was done to prevent namespace clash on similar elements in the xhtmlNPfITPresentationText schema imported into datatypes
    schema.

1.5        Added new schema NarrativeBlock.xsd

  •  Added NarrativeBlock CDA mark-up schema for Lab domain use.
