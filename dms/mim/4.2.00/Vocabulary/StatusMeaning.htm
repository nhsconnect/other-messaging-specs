Vocabulary for StatusMeaning

 

Description: Assigned OID:  2.16.840.1.113883.2.1.3.2.4.17.145  

 

A code which provides the meaning of the status display value.

 

Version:

01

Date:

12th October 2006

 

Values:

Code

Description

Note

1 To Be Resolved Indicates AR (Additional Requirement) type is in need of resolution
2 Not Provided Indicates AR type will not be provided

3

Booked

Indicates AR type is booked
4 Cancel, No Longer Required Indicates AR type was booked and is no longer required
5 No Longer Required Indicates AR type was not booked and is no longer required
6 Resolved Indicates the AR type was booked, but then the appointment was cancelled and the AR has now also been cancelled
