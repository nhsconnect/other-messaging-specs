Vocabulary for AdministrationSnCT

 

Description:

Assigned OID:  the OID for the UK subset of SnomedCT, q.v.

Codes taken from the UK subset of SnomedCT representing the administration of a single dose, cycle or complete course of medication to a patient, and its status (e.g. planned, done, abandoned).  If required, it may indicate the route(s) (e.g. intravenous) and/or the site(s) of administration (e.g. left arm). 

 

Legend:

A code of TBA indicates that the SnCT code is not yet available.

+ after a code indicates that descendants of the code shall be used.

The status of the medication administration

 

The status of an administration that has not yet been started but that it is intended to perform, e.g. it has been approved or scheduled.

 

The  status of an administration that it has been decided not to perform, e.g. it has been cancelled.

 

The  status of administration that has been started but not yet ended, i.e. it is ongoing or has been suspended. This can only apply to a course or cycle, or a single dose that takes a significant time to administer (e.g. to an infusion).

The  status of an administration that has been performed as planned.  It says nothing about the success or otherwise of the administration..
