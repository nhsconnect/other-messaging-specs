Vocabulary for DiscontinueMedicationTypeSnCT

 

Description:

Assigned OID:  the OID for the UK subset of SnomedCT, q.v.

Codes taken from the UK subset of SnomedCT representing that a medication being administered to a patient should be discontinued. 

 

Legend:

A code of TBA indicates that the SnCT code is not yet available.

+ after a code indicates that a descendant of the code shall be used.
