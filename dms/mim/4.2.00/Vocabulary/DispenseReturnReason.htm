Vocabulary for DispenseReturnReason

 

Description:

Assigned OID:  2.16.840.1.113883.2.1.3.2.4.16.28

Describes reasons for returning a prescription rather than dispensing it.

 

This will generally be present in an Observation act where the code attribute has an id value of “RR” from the Prescription Annotation vocabulary.
