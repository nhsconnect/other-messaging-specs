O5.1. Investigation - Interpretation ranges

The interpretation range for the results for each investigation result may represented using the ObservationStatement/referenceRange as shown in the table below. This segment is attached to the instance of the observation (ObservationStatement)  containing the investigation result.

ObservationStatement/referenceRange (ObservationStatement/referenceRange)

Refers to an interpretation range for an observation result.

Note

  • The primary intended use of this is for representation of laboratory investigation ranges in line with current UK clinical EDI requirements.

[1..1] typeCode (CS){ Fixed="REFV " }

Indicates that this is a reference range.

[1..1] referenceInterpretationRange (InterpretationRange)

Details of the reference range.

[1..1] classCode (CS){ Fixed="OBS " }

Indicates that this is a clone of the class observation

[1..1] moodCode (CS){ Fixed="EVN.CRT " }

Indicates that this represents criteria applied to an observation event.

[0..1] text (ED)

This is used for textual reference ranges, such as complex reference range information in NHS PMIP messages.

[0..1] value (IVL<PQ>)

This is used for numerical reference range values. The ranges may use any of the flavours of IVL<PQ> but are in most cases would use the following flavours:

  • Quantity Range
  • Quantity Greater than
  • Quantity Less than

The units used for the physical quantity in any of these flavours must be the same as that used for the approved unit of the result.
