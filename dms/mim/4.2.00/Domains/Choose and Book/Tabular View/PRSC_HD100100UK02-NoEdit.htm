CancelledSlotList

The Cancelled Slot List is a list of slots in a clinic that are no longer available for booking.

Used by:
[1..1] classCode (CS {CNE:LIST} ){ Fixed="LIST " }

Indicates that the cancelled slots take the form of a list.

  • Fixed value: "LIST"
[1..1] moodCode (CS {CNE:PRMS} ){ Fixed="PRMS " }

Indicates that this is a promise.

  • Fixed value: "PRMS"
[1..1] id (II)

A unique identifier for this instance  of a cancelled appointment list.

The Identifier Global datatype flavour is used to carry this information as follows:

  • A UUID is held in the root attribute
