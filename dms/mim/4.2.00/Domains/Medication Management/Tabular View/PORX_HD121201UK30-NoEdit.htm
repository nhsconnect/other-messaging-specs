PrescriptionReleaseReject

This message is used by the ETP Component to notify a dispensing system that its request for release of dispensing information has been rejected.  It can be used if the Release Request is either a Patient Prescription Release Request or a Nominated Prescription Release Request.

Reference: W5 Section 10.4.3.1
