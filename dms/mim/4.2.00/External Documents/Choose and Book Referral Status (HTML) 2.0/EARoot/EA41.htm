public int
  enquirerNHSNumber		 Range:0 to 1
Notes: The NHS number of the patient making the request. This value is only provided if the enquier is a patient.

If provided, normally this will be the same as the patient's NHS number but in the case of a proxy acting on behalf of a patient, it will be different.

An NHS Number or Role Profile code for the enquier must always be provided.
