The initialised request is an abstract class modelling associations common between appointment requests and advice and guidance requests.

In the CIM, an initialised request has a 0..1 relationship to activeServiceShortlist as this is not returned in the outline message.

An initialised request has a 0..1 relationship to searchCriteria as these are not returned in the outline message and are only returned in the detail message if clinical information is returned.
