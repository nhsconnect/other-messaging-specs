The structured referral letter is clinical information.

The UBRN, patient and priority code are not present in this class as they can be inferred from the request class. These fields may be added in order to aid reuse.

The structuredReferralLetter and the classes below are intended to model the retrieval of the entire contents submitted using the Referral message (PRPA_MT030101UK05).
