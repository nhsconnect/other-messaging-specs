public string
  text		 Range:0 to 1
Notes: Text specifying the nature of the procedure.

MIM states: "Text shall only be used if no code value is available and SHALL NOT be used in conjunction with Procedure Commentary."

The intention is for the text to be an alternative to the coded description. Regardless of whether a code or text is used to describe the procedure, the procedure commentary can be provided. However, the text, if provided, shall not be used to contradict or describe the same information as the procedure commentary.
