A request may store the original reason for referral, which was typically passed from the GP system with the Request Service message. If this was passed as a Read value, the code value may not be present, and the text will be returned. If this has been verified by the user and converted to a SNOMED CT term, this will return a coded value with its textual representation.

The reason for referral is clinical information.
