A clinical term is a coded value in a coding schema such as SNOMED used for a variety of purposes within a referral. The textual representation of the term is returned for information.

The clinicalTerm is clinical information.

Attribute Details
public int
  codeValue		 Range:0 to 1
Notes: The code value of the keyword. If a code value is provided then the keywordText should be the textual representation of the code value as defined by the coding schema. The coding schema must be provided if the code value is provided.
public string
  codingSchema		 Range:0 to 1
Notes: An identifier of the coding schema used for the keyword. E.g. SnomedCT or Read. A keyword could also be free text entered by the user or specified from a standard list maintained by Choose and Book. In the case of text then no coding schema would be specified.

Choose and Book will allow the text of a Read code to be supplied without its corresponding code value.

If a coding schema is specified, the a code value must also be provided.
Constraints:
List of values - MiM vocabulary

  keywordText
Notes: The text of the keyword. Either free text or if a code value is provided then this text should be the textual representation of the code value as defined by the coding schema.
