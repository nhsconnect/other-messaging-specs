Change Title:

 FORMTEXT PDS Batching changes

 

issue Description:

 FORMTEXT The infrastructure batching mechanism is changing in MIM 4.1.02. As a result, the PDS interactions that can be batched need to change to accommodate the new mechanism. (This supersedes MIM-CR-0696)

 

Proposed solution

 FORMTEXT Change the following interactions to remove the use of optional sender/receiver interactions:

PDS Trace Query Started - QUPA_IN010000UK32

PDS Trace Query Successful - QUPA_IN030000UK32

PDS Retrieval Query Started - QUPA_IN040000UK32

PDS Retrieval Query Successful - QUPA_IN050000UK32

 

Change the following interaction as a result of the above changes:

PDS Advanced Trace Query Started - QUPA_IN020000UK31

 

actual solution:

 FORMTEXT      

 

potential IMPACT if change not made:

 FORMTEXT High - batched PDS messages will be in the wrong format if the change is not made.

 FORMTEXT MIM 4.1.02
