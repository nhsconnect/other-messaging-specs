Short Title of Issue

DI Request to hold the Unique Booking Reference Number from Choose and Book. New DI Request Interaction and Trigger Event.

Description

There is a possibility that the Choose & Book processes may be used to ‘book’ an appointment for one or more DI Procedures known to require them. In order to facilitate this without significant change to the current design, the Unique Booking Reference Number, UBRN will be optionally included within the DI Request communication specification.

 

Bugzilla Issue 14867 - no documented way to withdraw a DI Request.

Perceived Impact

DI Request message: RMIM change, Tabular View, Schema and Examples.  Change to vocab TextType, to include BookingReference.
