Change Title:

Tabular Views for CMETs require further description.

 

issue Description:

CMET-TV changes required for MIM 4.2.00 and MIM 6.0.00.

 

Proposed solution

Description of CMETs in Tabular Views has been updated to clarify or modify their intended use. The CMET schemas have not been modified.

 

actual solution:

Descriptions to the following ‘clinically-related’ CMET-tabular views have been modified.:

A_SocialContext - UKCT_MT144036UK01

A_CareEvent - UKCT_MT144037UK01

A_Problem - UKCT_MT144038UK02

A_LinkAssertion - UKCT144039UK02

A_RiskToPerson - UKCT_MT144041UK01

A_Diagnosis - UKCT_MT144042UK01

A_Finding - UKCT_MT144043UK02

A_FamilyHistory - UKCT_MT144044UK01

A_PersonalPreference - UKCT_MT144046UK01

A_Service - UKCT_MT144047UK01

A_AllergyOrAdverseReactionDE - UKCT_MT144048UK02

A_ProvisionOfAdviceAndInformation - UKCT_MT144049UK01

A_AdministrativeProcedure - UKCT_MT144050UK01

A_AllergyOrAdverseReactionIdent - UKCT_MT144053UK02

A_Treatment - UKCT_MT144055UK01

 

NOTE: The actual CMET schemas remain unchanged.

 

potential IMPACT if change not made:

Less understanding of the use of clinical CMETs.

 

target release:

CMET Version 5.00 (MIM4.2.00 & MIM 6.0.00).
