Change Title:

GP Summary, Vocab & CMET 4.1.03 to 4.1.04 changes

 

issue Description:

Various issues raised resulting from the  release of MIM 4.1.03

 

Proposed solution

As discussed at supplier meeting.

 

actual solution:

1. Schema's and examples updated to now allow and include all fixed values.

STRUCTURAL CHANGES TO MESSAGES

2. UKCT144039UK02 was UK01 - A_LinkAssertion -
Link assertion has been remodelled in line with the latest problem representation.

The errors in InversionInd values have been corrected

The  ActRef constraints have been reviewed and updated

The CMET has also been made simpler and easier to understand by.

•        The use of helpful classnames.

•        The use of source and target in the name of the clinical statement link assertion

•        The use of problem member in the problem member assertion

3. UKCT_MT144038UK02 was UK01 A_Problem
The problem CMET now has been remodelled with a reference to the clinical statement which is identified as a problem. This aligns the CMET with the new problem representation.

 

4. UKCT_MT144048UK02 was UK01 A_AllergyOrAdverseReactionDE
The choice of allergy or adverse reaction acts have been amalgamated into one allergyDE act. Vocabularies have also been amalgamated into new vocabulary called AllergyOrAdverseReactionDESnCT.

5. UKCT_MT144053UK02 was UK01 A_AllergyOrAdverseReactionIdent
The choice of allergy or adverse reaction acts have  been amalgamated into one allergy act. Vocabularies have also been amalgamated into new vocabulary called AllergyOrAdverseReactionIdentSnCT.

 

6. UKCT_MT144043UK02 was UK01 A_Finding
The representation of blood pressure has been changed to align with the Clinical statement pattern.

•        id and effectiveTime added to SystolicPressure

•        id and effectiveTime added to DiastolicPressure

 

7. REPC_RM150002UK05 was UK04 GPSummaryText
- Removal of message replacement act and relationship within RMIM

8. REPC_MT150007UK05 was UK04 GPSummary
- changes and change to message replacement act within RMIM.


CHANGES TO CMET EXAMPLES

•        A_AdminstrativeProcedure-Administrative Procedure          Code changed.  SupportingInformation removed.

•        A_Diagnosis-Diagnosis with linked Finding    Example added with linked Finding to support LinkAssertion example.

•        A_Diagnosis-Diagnosis          Example added to support LinkAssertion example.

•        A_Finding-Blood Pressure     Code changed to reflect sitting blood pressure.

•        A_Finding-Temperature       Unit changed to "C" to reflect correct UCUM code.

•        A_Finding-Clinical Finding      Example added to support LinkAssertion example.

•        A_FamilyHistory-Family History          SupportingInformation removed.

•        A_Investigation-Investigation         Code amended.  Specimen and SupportingInformation removed.

•        A_LinkAssertion-Problem Link Assertion      References changed to provide full example based on other CMET examples.

•        A_PersonalPreference-Personal Preference Blank SupportingInformation removed as unnecessary.

•        A_Problem     Problem         2 examples added to support LinkAssertion example.

•        A_ProvisionOfAdviceAndInformation-Advice and Information    Code changed to code form subset.

•        A_Service-Service     Code changed to code from subset.

•        A_SocialContext-Alcohol Intake       Code changed to valid code from subset.

•        A_SocialContext-Smoking     Replace d-1 with 1/d to reflect correct UCUM units.
