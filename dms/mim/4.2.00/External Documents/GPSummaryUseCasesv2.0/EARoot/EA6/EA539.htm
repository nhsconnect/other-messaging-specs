01. Print an Initial GP Summary, Prior to Upload Constraint
Constraint Type Status Detail
Produce a printed summary Goal Approved To produce a copy, either printed or on screen, of what will be included in the patients Initial GP Summary.
The purpose of this summary is to enable the patient to see the contents of the summary that is going to be sent to the NHS Summary Record
Creation of a printed Initial GP summary Description Approved The Use Case depicts the Administrator creating an Initial GP Summary on screen and printing it out so that the patient can view what is going to be included in the Summary when it is sent to the NHS Summary Record
Administrator receives request to create printed summary Trigger Approved The administrator has received a request to create a printed Initial GP Summary for a specific patient
Patient registration, NPfIT compliance, LRs Pre-condition Approved Refer to GP Summary, Guidance for Suppliers, NPFIT-SHR-MODL-SUMREC-0008.(current)
A view of the patients InitialGP Summary will have been created Post-condition Approved A copy or view of the summary which is going to be sent to the NHS Summary Record
What to include in the Initial GP Summary Business Rules Approved Refer to GP Summary, Guidance for Suppliers, NPFIT-SHR-MODL-SUMREC-0008.(current)
