6.3.00 25/09/2007 Changes consist of :
  • Changes required for CCN404 ( Detailed Technical Specification Version 0.7 )
  • Changes to Templates
    • New Encompassing encounter template used in
      1. Discharge
      2. Outpatients
      3. Emergency Department
    • New datatype flavour for all codes within coded entry templates
    • Removal of Business sections and CRE type section templates
    • Removal of all specific text section templates and replacement by 1 generic text section template for all domains
    • Changes to the classification of coded entries using CREtype

  • Generic model CDA model has been updated to align with CCN 404 spec
  • Sealed Envelopes - minor changes in addition to changes required for CCN 404 spec
  • PSIS query changes as per CCN 404 spec

6.1.01 30/03/2007 The changes included in this release are documented in the change control section on the relevant IM documents.

 

Alerts - New Domain - Generic Alert message, first usage is to be within Sealed Envelope processing.

  • XML Examples have been reorganised within the Examples directory.

  • CMETS, Vocabulary, OIDs and Templates IM files have been reorganised.

  • The OIDs catalogue has been relocated to the "Domains - Common" folder within the navigation menu.
