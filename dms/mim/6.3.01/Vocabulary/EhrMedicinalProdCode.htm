Vocabulary for EhrMedicinalProdCode

Version: 4.0
Status: active
Date: 2005-03-22

Description

Clinical code or product identification identifying a virtual or actual medicinal product or product pack.

Code representing a prescribable drug or product, taken from one the following code systems:

SNOMED CT UK
A code from Snomed-CT (UK) may be accompanied by additional qualifying codes from the same coding system. In future should adopt NHS PCDD identifiers once they are in general use.
Read Codes V2
EMIS Drug Codes
Another code system, e.g. Multilex
