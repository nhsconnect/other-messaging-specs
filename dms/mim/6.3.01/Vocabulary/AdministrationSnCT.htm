Vocabulary for AdministrationSnCT

Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 2.0
Status: active
Date: 2007-04-16

Description

A subset of SNOMED-CT intended for use in the UK representing the administration of a single dose, cycle or complete course of medication to a patient, and its status (e.g. planned, done, abandoned). If required, it may indicate the route(s) (e.g. intravenous) and/or the site(s) of administration (e.g. left arm).

Any code from the SNOMED CT UK Administration Medication subset with subset original id 1311000000134.

The use of this subset is Mandatory as defined in latest version of "NPFIT-ELIBR-AREL-P1R2-0178 Technical Guidance for Implementation of Templated CDA Domains."
