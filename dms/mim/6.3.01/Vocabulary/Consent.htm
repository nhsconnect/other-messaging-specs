Vocabulary for Consent

Code System Id

2.16.840.1.113883.2.1.3.2.4.16.2

Version: 4.0
Status: active
Date: 2004-06-25

Description

Default Scheme: NHS Data Dictionary Consent

Assigned OID: 2.16.840.1.113883.2.1.3.2.4.16.2

A patient's consent to having their clinical details used by NHS organisations for clinical consultation, treatment and for secondary uses of the data such as research purposes. There are also codes that give the status of consent to being called back from a call centre.

Values

Code Description Note
1 Express consent

Does not apply to call centre call back consent.

NB: The description has been modified from "Consent given" since P1R1 release as there has been an agreed change of emphasis relating to consent codes.
2 Express dissent

Does not apply to call centre call back consent.

NB: The description has been modified from "Consent not given" since P1R1 release as there has been an agreed change of emphasis relating to consent codes.
4 No

Only applies to call centre call back consent
5 Yes

Only applies to call centre call back consent
