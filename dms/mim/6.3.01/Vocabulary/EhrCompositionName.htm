Status: active
Date: 2005-10-24

Description

Assigned OID : 2.16.840.1.113883.2.1.3.2.4.15

Set of composition names based on a superset of types already in use.

Note that if events such as repeat prescription issue, investigation requests, report filing occur within a consultation the entire interaction is recorded in a composition with the name applicable to that type of consultation note. The specific prescription issue, report and request compositions are intended for use only where these interactions occur without a direct patient contact or separated in time from such a contact.
