Description

A unique code representing a constituent internal part of the United Kingdom of Great Britain and Northern Ireland.

This is a controlled subset of the UK Government Data Standards Catalogue UK Internal Code code system (see http://www.govtalk.gov.uk/gdsc/html/frames/UKinternalCode.htm).
