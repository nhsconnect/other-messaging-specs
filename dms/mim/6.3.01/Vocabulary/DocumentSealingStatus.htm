Vocabulary for DocumentSealingStatus

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.179

Version: 1.0
Status: active
Date: 2006-12-18

Description

Indicates whether the document contains sealed information.

Values

Code Description Note
0 No sealed content

This is used when the document contains no sealed content at all
1 Sealed content

This is used when the document contains some sealed content at entry and/or text level
2 Document sealed

The whole document is sealed
