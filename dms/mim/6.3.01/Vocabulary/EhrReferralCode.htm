Vocabulary for EhrReferralCode

Version: 2.0

Date: 2004-03-22

Description

A concept from a controlled clinical terminology indicating the reason for referral or the nature of a requested service.

A code representing a kind of referral or requested service, taken from one of the following code systems:

SNOMED CT UK
A code from Snomed-CT (UK) may be accompanied by additional qualifying codes from the same coding system.
Read Codes Version 2
Another code system.
