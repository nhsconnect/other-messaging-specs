Vocabulary for PrescriptionStatus

Code System Id

2.16.840.1.113883.2.1.3.2.4.16.35

Version: 3.0
Status: active
Date: 2005-05-17

Description

Default Scheme: NHS Data Dictionary TBA

Assigned OID: 2.16.840.1.113883.2.1.3.2.4.16.35

Describes the status of a prescription or of an item on a prescription.

This will generally be present in an Observation act where the code attribute has an id value of "PS" from the Prescription Annotation vocabulary.

Values

Code Description Note
0001 To be Dispensed
0002 With Dispenser
0003 With Dispenser - Active
0004 Expired
0005 Cancelled
0006 Dispensed
0007 Not Dispensed

When this value is used a second coding system is used to describe the non-dispense reason
