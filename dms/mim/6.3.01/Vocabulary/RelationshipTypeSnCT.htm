Vocabulary for RelationshipTypeSnCT

Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 3.0
Status: active
Date: 2005-07-15

Description

Assigned OID: the SnomedCT OID, qv the link below

Any code from the UK subset of SnomedCT that describes a relationship between the contents of one clinical statement and another, for example 'is manifestation of', 'has reason', 'has support'.

A code taken from SNOMED CT as specified in the most recently released version of the UK SNOMED CT subset with the SubsetOriginalId of 581000000132.

The code shall describe a type of relationship between one clinical statement and another

A code from SNOMED CT (UK) may be accompanied by additional qualifying codes from the same coding system.

Refer to section 5.2.4 of the Messaging Technical Guidance for further information on the specific SNOMED CT values to use.
