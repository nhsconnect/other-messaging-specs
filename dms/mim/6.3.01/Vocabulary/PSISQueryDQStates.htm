Vocabulary for PSISQueryDQStates

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.188

Version: 1.0
Status: active
Date: 2007-01-31

Description

Used for PSIS Query Request parameters, this vocab provides the different available collections of data-quality states that documents can be queried on.

Values

Code Description Note
ALL All DQ States

A set which includes all data-quality states currently stated in the DocumentDQState vocabulary.
NOTDQ Not DQ Nullified

A set which includes all data-quality states currently stated in the DocumentDQState vocabulary with the exception of DQNULLIFIED.
