Vocabulary for SDSJobRoleName

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.196

Version: 2.0
Status: active
Date: 2007-10-19

Description

Identifies the job role associated with the person.

This may contain any code which describes the person's job role.

In the absence of a local OID, the OID for this vocabulary may be used. 

It is not recommended that these codes are used for machine processing.
