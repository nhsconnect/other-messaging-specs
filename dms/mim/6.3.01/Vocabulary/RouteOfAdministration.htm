Vocabulary for RouteOfAdministration

Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 1.0
Status: active
Date: 2003-11-28

Description

A code for the route by which a clinical product (medication, dressing or application) is administered, e.g. intravenously by infusion. oral to swallow, inhalation.

Any Snomed-CT code that is a descendant of the concept "route of administration values"

The rubrics for the codes are based on the "Standard Terms for Pharmaceutical Dosage Forms" published by the European Pharmacopoeia Commission.
