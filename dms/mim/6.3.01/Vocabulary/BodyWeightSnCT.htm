Vocabulary for BodyWeightSnCT

Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 2.0
Status: active
Date: 2007-04-16

Description

A subset of the Snomed-CT clinical terminology (i.e. coding system) intended for use in the UK, previously named SnomedCTUK, that describes a finding that is a measurement of body weight.

Any code from the SNOMED CT UK Weight subset with subset original id 971000000138.
