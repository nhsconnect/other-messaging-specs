Vocabulary for ClinicalProductForm

Version: 2.0
Status: active
Date: 2004-04-05

Description

A code for the physical form in which a clinical product (medication, dressing or appliance) is administered, e.g. as tablet, lozenge, linctus, oral liquid, aerosol, bandage, stoma bag.

Code describing a medicinal product dosage form, taken from one the following code systems:

SNOMED CT UK
A code from Snomed-CT (UK) may be accompanied by additional qualifying codes from the same coding system.
Clinical Terms (Read Codes) v3
A code from Clinical Terms (Read codes) Version 3 may be accompanied by additional qualifying codes from the same coding system.
EMIS Drug Codes
