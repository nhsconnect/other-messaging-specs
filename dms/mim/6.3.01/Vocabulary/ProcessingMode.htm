Vocabulary for ProcessingMode

Version: 2.0
Status: active
Date: 2004-05-10

Description

Derived from: HL7 v3 ProcessingMode of Ballot Pack 6, Jan 2004

This code defines whether the message is being sent in current processing (the usual mode), archive mode, initial load mode, restore from archive mode, etc..

Values

Code Description Note
A Archive mode

Identifies archive mode of processing.
I Initial load

Identifies initial load mode of processing.
R Restore from Archive

Identifies restore mode of processing
T Current processing

Identifies on-line mode of processing
