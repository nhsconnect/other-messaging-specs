Vocabulary for DocumentDQState

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.189

Version: 1.0
Status: active
Date: 2007-01-25

Description

A vocabulary to specify the available data-quality states that a PSIS record can have attributed to it.

Values

Code Description Note
NORMAL Normal

Where a clinical judgement determines that the clinical event belongs in the Patient's clinical record.

DQSUSPECT Data Quality Suspect

Where a clinical judgement determines that it is either work in progress or not possible to identify whether the clinical event belongs in the Patient's clinical record. Clinical events marked as suspect should normally be viewable as part of the Patient's clinical record.
DQNULLIFIED Data Quality Nullified

Where a clinical judgement determines that the clinical event does not belong in the Patient's clinical record.
