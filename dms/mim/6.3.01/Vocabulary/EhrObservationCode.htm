Vocabulary for EhrObservationCode

Version: 4.0
Status: active
Date: 2005-03-22

Description

Any code from a controlled clinical vocabulary supported in the communicating community. This should include a Read Code from Version 2.

See also notes on coded use of EhrObservationValue.

Code taken from one the following code systems:

SNOMED CT UK
A code representing a kind of observation which has a separate expressed value of some type. Code may be accompanied by additional qualifying codes from the same coding system.
SNOMED CT UK
A code representing a disorder, finding or clinical event. Code may be accompanied by additional qualifying codes from the same coding system.
Read Codes V2
Another code system
Code representing an observation or finding.
