Vocabulary for SDSObservationType

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.131

Version: 1.0
Status: active
Date: 2006-05-02

Description

Assigned OID: 2.16.840.1.113883.2.1.3.2.4.17.131

A code indicating the type of observed information. The information is derived from SDS (Spine Directory Services)

Values

Code Description Note
01 Identified workgroups only

Used to indicate whether or not the User Role Profile has Workgroup Membership Administration rights for identified workgroups only (as opposed to, for example, all Workgroups in or linked to the Organisation within the User Role Profile).
02 Already a member of the identified workgroup

Used to indicate whether or not a User Role Profile was already a member of the identified Workgroup when an Add User To Workgroup request was submitted.
03 Was not a member of the identified workgroup

Used to indicate whether or not a User Role Profile was not actually a member of the identified Workgroup when a Remove User From Workgroup request was submitted.
