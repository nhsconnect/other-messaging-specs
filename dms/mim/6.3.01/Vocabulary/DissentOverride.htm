Vocabulary for DissentOverride

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.60

Version: 3.0
Status: active
Date: 2007-11-01

Description

A code indicating whether or not dissent to access a patient's NHS Care Record should be overriden, and if so, the reason for it. This vocabulary is a combination of previous vocabularies: DissentOverrideIndicator and DissentOverrideReasonCode.

Values

Code Description Note
0 No override
1 Access made in the public interest
2 Access required by statute
3 Access required by Court Order
4 Query for update

System-to-system use only, no information retrieved with this override code should be displayed to the user.
5 Override dissent with temporary patient consent
