Vocabulary for EhrActState

Version: 1.0
Status: active
Date: 2004-05-10

Description

Indicates whether a component or statement (represented by an Act) in the record is currently valid, deleted or replaced. Applies to any component in the patient record.

Values

Code Description Note
COMPLETE Currently valid

Complete implies a valid record that has been completed. This is distinct from other states in the HL7v3 state diagram.
NULLIFIED No longer valid and not replaced (i.e. marked as deleted)

Should be referenced by an empty act (EhrEmpty) instance that indicates the context in which it was deleted.
OBSOLETE Replaced or superseded by a corrected or more up to date statement

Should be referenced by the replacement statement.
