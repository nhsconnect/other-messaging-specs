Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 2.0

Description

A code that may be conveyed as a value associated with an observation to indicate that a problem or diagnosis is regarded as having special significance.

For example an alert.

Values

Code Description Note
386134007 Significant

Indicates that an item should be highlighted and/or included in a summary.
394847000 Not specified

Default value
