Vocabulary for RangeCriterionSnCT

Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 1.0
Status: active
Date: 2006-02-15

Description

This vocabulary is used to convey the Criteria that constrain the Interpretation Range. There may be multiple criteria applied to a quoted range.

The current scope provides an SnCT concept to which additional text may be added to convey details of the Range Criteria and for specific criteria related to the Reference Change Values that may be quoted.

Values

Code Description Note
220221000000104 Range criteria (qualifier value)

Allows additional descriptive text to be carried
220231000000102 Reference change value 95% probability of significance (qualifier value)

Identifies that the value carried in the InterpretationRange is the change (% or Absolute) above which there is a 95% probability of significance
220241000000106 Reference change value 99% probability of significance (qualifier value)

Identifies that the value carried in the InterpretationRange is the change (% or Absolute) above which there is a 99% probability of significance
