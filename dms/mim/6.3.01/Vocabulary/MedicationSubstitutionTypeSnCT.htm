Description

Assigned OID: the OID for the UK subset of SnomedCT, q.v.

Codes taken from the UK subset of SnomedCT representing that a different medication should be substituted for an existing medication.

Legend:

A code of TBA indicates that the SnCT code is not yet available.
+ after a code indicates that descendants of the code shall be used.
