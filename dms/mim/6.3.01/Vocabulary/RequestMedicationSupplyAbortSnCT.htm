Vocabulary for RequestMedicationSupplyAbortSnCT

Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 1.0
Status: active
Date: 2005-01-13

Description

Assigned OID: the OID for the UK subset of Snomed-CT, q.v.

Codes taken from the UK subset of SnomedCT representing a a request to abort a pharmaceutical product supply request (e.g. NHS prescription, hospital prescription).

Legend:

A code of TBA indicates that the SnCT code is not yet available.

+ after a code indicates that descendants of the code shall be used.

Values

Code Description Note
