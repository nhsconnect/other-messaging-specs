Description

Default Scheme: NHS Data Dictionary TBA

Assigned OID: 2.16.840.1.113883.2.1.3.2.4.16.29

Describes the possible endorsements that a dispenser may apply to a prescription.

This will generally be present in an Observation act where the code attribute has an id value of "DE" from the PrescriptionAnnotation vocabulary.

Note: The PPA systems are Euro compliant. Hence, the inclusion of endorsement codes for pence or cents.
