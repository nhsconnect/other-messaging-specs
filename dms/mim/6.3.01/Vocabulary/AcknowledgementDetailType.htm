Description

Codes classifying the severity of the issue and carried by the AcknowledgementDetail.typeCode.

Values

Code Description Note
ER Error

The acknowledgement identifies an error.
WG Warning

The acknowledgement identifies an issue which is a warning.
IF For Information

The acknowledgement identifies an issue for information.
