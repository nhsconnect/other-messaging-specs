Description

Default Scheme: NHS Data Dictionary Discharge Destination

Assigned OID: 2.16.840.1.113883.2.1.3.2.4.16.40

A code representing the destination of a patient on completion of a Hospital Provider Spell, or a note that the PATIENT died or was a still birth. This is the most appropriate vocabulary that currently supports the disposition/discharge described in the ED & OOH domains. This vocabulary needs to be refined.
