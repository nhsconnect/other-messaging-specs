Vocabulary for EhrGP2GPClusterCode

Version: 2.0
Status: active
Date: 2005-01-07

Description

This value set only applies to CompoundStatements at 'cluster' level (classCode='CLUSTER').

It is used to name subdivisions that group together a logically connected set of observations and circumstances applicable to those observations.

A code representing the name of a cluster of statements, taken from one of the following code systems:

SNOMED CT UK
A code from Snomed-CT (UK) may be accompanied by additional qualifying codes from the same coding system.
Read Codes V2
Another code system.
