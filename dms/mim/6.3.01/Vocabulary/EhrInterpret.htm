Vocabulary for EhrInterpret

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.304

Version: 1.0
Status: created
Date: 2004-05-10

Description

Indicates an interpretation by the originator of an observation.

Uses the code set used in NHS PMIP project. There is no "Normal" in this set as it was felt to be inappropriate to state normality as a positive assertion. The interpretationCode field is optional and is omitted unless one of the specified values applies.

Values

Code Description Note
HI Above high reference limit
LO Below low reference limit
OR Outside reference range
PA Potentially abnormal
