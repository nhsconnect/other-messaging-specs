Vocabulary for x_DosageMood

Version: 1.0
Status: active
Date: 2004-05-10

Description

Distinguishes between a dosage instruction and actual administration of a dose.

See also x_MedicationMood.

Values

Code Description Note
EVN Event

A dose actually administered or observed to be administered.
RMD Recommendation

A dosage instruction (recommended, prescribed or authorised).
