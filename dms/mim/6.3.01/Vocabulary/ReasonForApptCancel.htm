Version: 3.0
Status: active
Date: 2007-04-25

Description

Default Scheme: NHS Data Dictionary ReasonForApptCancel

A classification of why the appointment was cancelled. Generally codes 01-19 would be used in the Notify Appointment Cancellation Message, and codes 20-39 in the Request Appointment Cancellation message.

The concepts defined by this code system are maintained externally by Choose and Book.
