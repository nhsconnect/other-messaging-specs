Description

A subset of SNOMED-CT intended for use in the UK, that describes the lifestyle of the patient.

Any code from the SNOMED CT UK Lifestyle subset with subset original id 921000000137.

The use of this subset is Advisory as defined in latest version of "NPFIT-ELIBR-AREL-P1R2-0178 Technical Guidance for Implementation of Templated CDA Domains."
