Vocabulary for AdministrationSiteSnCT

Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 1.0
Status: active
Date: 2004-06-25

Description

Assigned OID: the OID for the UK subset of SnomedCT, qv the link below

Describes the place on the body by which a medicine enters the body. For instance a tablet has an approach site of "by mouth".

The vocabulary should be drawn from the relevant part of Snomed CT.

Was SnomedActSite prior to release 3.1.

Any Snomed-CT code that is a descendant of the concept "Physical Anatomical Entity".

This area of coding is not well defined. Further work is underway to develop a better defined hierarchy.
