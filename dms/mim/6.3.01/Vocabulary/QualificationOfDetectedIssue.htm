Vocabulary for QualificationOfDetectedIssue

Version: 1.0
Status: active
Date: 2007-01-15

Description

This vocabulary formally specifies how the code systems identified under ActDetectedIssueCode shall use qualifiers from the DetectedIssueQualifier code system. One of the these qualifiers shall be used to qualify every code from each code system carried in Trigger Event Control Act and Query Acknowledgement Response wrappers DetectedIssueEvent.code.
