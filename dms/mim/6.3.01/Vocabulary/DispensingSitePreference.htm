Description

Assigned OID: 2.16.840.1.113883.2.1.3.2.4.17.21

Describes the nature of the role played by an agent (in this case a pharmacy) when supplying medication.

Note:1) Message coding reflects the PDS code scheme for P1R2.

2) The role of Agent is a place holder, not an actual specified pharmacy
