Vocabulary for CDAPrescriptionAnnotation

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.192

Version: 1.0
Status: active
Date: 2007-03-09

Description

Describes the type of annotations and observations that may be made to a prescription either to convey incidental information for reimbursement, or to manage the business processes around prescriptions. It also includes several values which are for use in other kinds of medication statements, which may at some time in the future become a separate vocabulary.

Values

Code Description Note
SI Supply Instructions

This has no related table of coded values.

It indicates that an observation will contain a free text entry providing instructions that the dispenser will provide to the patient.
DI Dose Instructions

This has no related table of coded values.

It indicates an observation that contains a free text description of the dosage instructions for a medicine. For instance "ONE tablet to be taken 3 times a day before meals"
AI Additional Instructions

This has no related table of coded values.

It indicates that an observation will contain a free text entry providing instructions that the prescriber will provide to the patient.
AD Medication administration Instructions

Categorises supporting information used by medication statements.
AT Medication administration type

Categorises supporting information used by medication statements.
ASF Acute Script Flag

Indicates whether the prescription was intended to be a one off prescription.
