Vocabulary for Determiner

Version: 1.0
Status: active
Date: 2004-05-10

Description

Indicates whether an entity represents a kind of or a specific instance.

Values

Code Description Note
KIND Described

To indicate that the given Entity is taken as a general description of a kind of thing that can be taken in whole, in part, or in multiples.
QUANTIFIED_KIND Described quantified

Indicates that the given Entity is taken as a general description of a specific amount of a thing. For example, QUANTIFIED_KIND of syringe (quantity = 3,) stands for exactly three syringes.
INSTANCE Specific

Indicates that the given Entity is taken as one specific thing instance. For example, a human INSTANCE (quantity = 1,) stands for exactly one human being.
