Vocabulary for AllergyOrAdverseReactionDESnCT

Version: 2.0
Status: active
Date: 2007-09-11

Description

A subset of SNOMED-CT intended for use in the UK, that describe diagnosis events of allergies or adverse reactions.

Refer to the latest version of "NPFIT-ELIBR-AREL-P1R2-0178 Technical Guidance for Implementation of Templated CDA Domains."
