This class exists to concretise the abstract initialisedRequest in the case that the request is for advice and guidance.

An outline message or a detail message that does not include clinical information will require an indication that the request is for advice and guidance but cannot return the adviceAndGuidanceRequestDetails class as this contains clinical information.
