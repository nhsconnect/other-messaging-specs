public int
  referralCategoryHeader
Notes: Category header must have one of the following values according to the ReferralCategoryHeading vocabulary from MIM 4.1.0:
Problem being referred
Investigations
Examinations
Current medication
Allergies
Intolerances
Risk to others
Alcohol consumption
Smoking habit
Non-therapeutic drug use
Conditions
Diagnoses
Procedures
Medication
Family history
Housing
Employment

Section 7.12 of the e Booking domain in MIM 4.1.0 describes a set of referral category headers similar to the above with the addition of Religion, Ethnic Origin, Language and Consultations.

We have chosen to use the set of values from the vocabulary as only values from this set can be submitted.
