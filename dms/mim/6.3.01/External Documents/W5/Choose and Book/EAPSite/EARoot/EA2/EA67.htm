If the patient was stop-noted with I, this will not result in a successful PDS retrieval. If patient is stop-noted with B, patient details may be retrieved, the value B will be returned to the calling application.

This check is made before further verifying if the user is a patient and is entitled to see the requested information because if the patient is stop-noted, PDS will not return sufficient information to verify this completely. Hence if the patient is stop-noted, there is no point trying to check whether the user is the patient.
