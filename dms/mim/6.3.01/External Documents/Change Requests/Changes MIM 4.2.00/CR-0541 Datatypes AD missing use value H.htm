Short Title of Issue

Value H is missing from use attribute values for AD datatype

 

 

Description

This relates to issue #13056 where some of the PDS tabular views reference value "H" for the use attribute in the AD datatype, but the datatypes document lists several use attribute values, but not "H". "H" is a legitimate value and needs to be added into the datatypes document to remove this inconsistency.

 

 

Perceived Impact

See Key below

Should not affect other domains. Potentially significant impact as suppliers will not know whether PDS messages are legitimately constructed.

 

Backward Compatibility Status

 

Reason for issue being raised

Issue #13056 raised by BT NASP.

 

 

Suggested Action

Amend datatypes document as suggested above.
