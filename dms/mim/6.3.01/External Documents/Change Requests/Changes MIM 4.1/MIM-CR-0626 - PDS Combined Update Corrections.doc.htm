Short Title of Issue

Corrections to PDS Combined Update message for inclusion in 2006B MIM release.

 

 

Description

The PDS Combined Update example messages contain a number of minor errors carried over from the PDS General Update message, notably:

  • Uses a code of 3 to indicate callback centre consent - should be 6.
  • Contains the removal of BirthTime - which is not permitted.

 

In addition, on the RMIM diagram the Consent Act does not link to the tabular view. This needs to be corrected.

 

Also, the RMIM diagram would benefit from a text note indicating that effective time should not be populated for callback consent, althoug hthe tabular view does already state this.

Perceived Impact

No impact on other domains. These are minor corrections where:

  • some content of example messages deviates from requirements stated in the tabular view;
  • there is a missing hyperlink from the RMIM to tabular view;
  • the RMIM diagram would benefit from extra clarifying text, although this text is already to be found in the tabular view.

 

Backward Compatibility Status

 

Reason for issue being raised

Issues raised in comments by BT NASP on the new PDS Combined Update message added into MIMv3.1.11.

 

Suggested Action

Make amendments to example messages and tabular view as suggested above.
