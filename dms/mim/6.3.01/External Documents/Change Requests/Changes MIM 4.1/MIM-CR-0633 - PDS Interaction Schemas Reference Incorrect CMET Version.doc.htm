Short Title of Issue

PDS Interaction Schemas Reference Incorrect CMET Version

 

 

Description

All the PDS interaction schemas reference an incorrect version of one of the CMETs that is found in both the Send Message Payload and Application Acknowledgement Transmission wrappers: UKCT_MT120501UK02.xsd. Although this schema is referenced, the CMET found in the RMIM diagram is UKCT_MT120501UK03, therefore the correct schema reference should be to UKCT_MT120501UK03.xsd. However, the UKCT_MT120501UK02 CMET was never referenced in any published version of these Transmission wrappers.

 

As these interaction schemas have been generated using incorrect HMD files, the safest course of action is to regenerate all the affected schemas against the correct HMD files. This will require reversioning of the affected interactions. For MIMv4.1, 5 of the now remaining 11 interactions have already been reversioned for other reasons. Therefore this issue will effectively require reversioning to be applied to an additional 6 interactions.
