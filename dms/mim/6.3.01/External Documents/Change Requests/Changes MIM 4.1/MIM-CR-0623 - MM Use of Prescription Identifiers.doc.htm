Short Title of Issue

Use of Prescription Identifiers

Description

 

Bugzilla issue

 

14108

Guidance not provided for the usage of the message for either the 18 character or the 36 character GUID. Which one to use when? PrescriptionID to be used for

14237

MIM-CR-0382 - MM 18 character prescription ID 

1.        Prescription Response does not state 18char length for Prescription id 

2.        PA Notification, PA Notification with Claim Information, and PS Notification with Claim Information do not have a Prescription ID.

Why are they listed in the CR?

Perceived Impact

Tabular view or vocab changes required.

Reason for issue being raised

This CR was raised been following the BT NASP review of MiM 4.0.01 and supersedes MIM-CR-0382 in addressing the use of prescription identifiers to ensure backwards compatability between MM v1 and v2 messages.

Suggested Action
  1. Where the tabular view within the PrescriptionID act currently says this;

Prescription identifier that is universally unique across all records and consists of either:

·                                 the globally unique number (DCE UUID) that identifies the Prescription clinical event.  

The Identifier Global datatype flavour is used to carry this information as follows:

o                                the root attribute shall contain the value of the DCE UUID

or

·                                 the Short Form Prescription Identifier (Barcode GUID).

The Identifier External datatype flavour is used to carry this information as follows:

o                                the root attribute shall contain the value of the Short Form Prescription Identifier OID = 2.16.840.1.113883.2.1.3.2.4.18.8

o                                the extension attribute shall contain the 18 character Short Form Prescription Identifier

Please change it to this;

The prescription identifier shall be the Short Form Prescription Identifier where this exists.

The 36 character clinical statement DCE UUID of the prescription shall only be used where the Short Form Prescription Identifier does not exist.
