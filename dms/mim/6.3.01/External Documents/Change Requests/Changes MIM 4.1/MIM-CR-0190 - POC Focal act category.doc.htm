Use of MIM Change Request:

 

The MIM Change Request shall be used for all changes required to the MIM that are not part of a W5 Change request.

This affects all changes to the Models/Implementation Manual/Tabular Views/Schemas/Examples etc that fall outside the requirements defined in the W5. This includes any required change to the Clinical Statement Message Pattern (CSMP).

 

All changes shall be submitted to the CTT Manager for review and submission to the Project Board once the impact has been estimated.
