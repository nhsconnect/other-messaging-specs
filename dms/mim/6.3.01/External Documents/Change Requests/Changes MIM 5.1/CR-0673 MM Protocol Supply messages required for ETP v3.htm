Short Title of Issue

Protocol Supply messages required for ETP v3.

Description

 

Bugzilla issue

 

15750

 
The messages supporting the supply of medication by pharmacies under patient Group Directives and Minor Ailments Schemes (generally known as Protocol Supply) are changing to convey additional business information to the PPA.
 
These changes require reinstatement of the PS message set as is.

Perceived Impact

PS artefacts to be reinstated as last published in MM 2.0, except for PS Notification, PS Claim Information and PS Notification with Claim Information.  These artefacts were updated and had there versions incremented, as MIM-CR-0636, although they did not appear in the following IM version (MM 2.1).

Reason for issue being raised

Enhancement request by ETP team as part of changes for ETP v3 / Spine release 2007-A. 

Suggested Action

Reinstate the following artefacts:
