Change Title:

Model and Tabular view and wording changes required following MIM 5.3 review

 

issue Description:

Following reviews of the PDS domain in MIM 5.3, a number of errors in the models and tabular views have been identified, and so a number of changes are required.

 

Proposed solution

Update the following models and corresponding interactions:

 

PDS Advanced Trace Query - QUPA_MT000002UK01 ( ParameterList and PatientCareProvision classes were removed for clarity. Minor changes to the wording of the tabular view, and changes to schemas and examples.)

 

PDS Advanced Trace Query Response - PRPA_MT000207UK01 (New choice class, "Source" containing classes: "SourceIdentified" and "SourceCoded". Removed null flavours from multipleBirthOrderNumber and clarified use in accordance with NHS Data Dictionary. Minor changes to the Example 1. New example 3 to show system generated effective dates and source.)

 

PDS Retrieval Query Response - PRPA_MT000201UK01 (New choice class, "Source" containing classes: "SourceIdentified" and "SourceCoded". Model change to PatientCareProvisionEvent.subjectOf1.SystemEffectiveDate.value becomes 1..1). Removed null flavours from multipleBirthOrderNumber and clarified use in accordance with NHS Data Dictionary. Minor changes to the Example 3. Added a new datatype flavor, AD.NPfIT.CodedCountry for the use of Birthplace.addr)

 

PDS Batch Trace Query - QUPA_MT000005UK01 (ParameterList and PatientCareProvision classes were removed for clarity. Minor changes to the wording of the tabular view, and changes to schemas and examples. )

 

PDS Cross Check Trace Query Response - PRPA_MT000213UK01

 

Regenerate tabular views for the above models:

 

PDS Simple Trace Query Response - PRPA_MT000200UK01 (Removed null flavours from multipleBirthOrderNumber and clarified use in accordance with NHS Data Dictionary. Minor change to the Example 1, removing City and Country elements of person address.)

 

PDS Batch Trace Query Response - PRPA_MT000209UK01 ( Changes to the tabular view descriptions to associate descriptions with correct attributes. Removed null flavours from multipleBirthOrderNumber and clarified use in accordance with NHS Data Dictionary. Minor change to example message 3. )

 

PDS Cross Check Trace Query - QUPA_MT000006UK01 (Changes to the tabular view descriptions to associate descriptions with correct attributes.)

 

Change IM file to include a new example 3 for  PDS Advanced Trace Query Response.
