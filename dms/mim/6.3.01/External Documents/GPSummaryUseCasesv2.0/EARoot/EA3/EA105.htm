Scenario Type Detail
Send GP Summary to NHS Summary Record Basic Path 1. Local System populates GP Summary message with relevant items
2. Local System sends GP Summary message to NHS Summary Record
3. Local System keeps a record of which items have been included and/or excluded from this GP Summary message
4. NHS Summary Record system acknowledges receipt of message
5 End.
