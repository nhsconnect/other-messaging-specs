Validate all recipient systems using an SDS Lookup.
Check for valid ASID and version information for compliance.

For all the recipients that cannot receive the message the user should be notified
and an alternative/manual process will be needed.

The message will only include a list of recipients who can actually receive the message (i.e. all those to whom the message is attempted to be send to via this process).
