Problem Template

Template used with a reference to a clinical statement to indicate that the clinical statement has been highlighted as a problem

Notes:
1. In the CDA Model every coded entry template will have a TextSection. In this class the Text and Title attributes from the TextSection have been merged/incorporated into this class,so a separate TextSection class does not clutter the diagrams.
2. Technical (i.e. message and HL7 specific) attributes have not been included.
