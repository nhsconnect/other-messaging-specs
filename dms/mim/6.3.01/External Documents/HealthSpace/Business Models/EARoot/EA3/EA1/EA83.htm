The PDS holds a number of different consent statuses for each service user. Examples include Consent to NHS Care Record sharing, Consent to Secondary Uses and Consent to eBS call centre callback. The Consent object contains the status, the date the status was changed and an optional comment for type of consent. The Person object may have zero to many associated Consent objects, but only one set of consent data may be stored for each consent type.
