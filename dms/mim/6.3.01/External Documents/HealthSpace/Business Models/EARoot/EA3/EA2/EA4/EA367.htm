Patient Entered HealthSpace Summary - A Healthspace user will be able to view information about his/her treatment preferences which he/she has entered into PSIS.
Examples of the sorts of records a Healthspace user may wish to enter are: Faith and spiritual requirements, Dietary Requirements, Disability support requirements, etc.

View HealthSpace Summary Constraint
Use Case Detail
1. Goal A Healthspace user will be able to view information about his/her treatment preferences which he/she has entered into PSIS
2. Trigger User wishes to view HealthSpace Summary data
3. Pre-condition User (Patient) authenticated to eGif Level 3
Authority to access required information and legitimate relationship
Existence of NHS care record for patient on PSIS/PDS/HealthSpace
4. Success Post-condition HealthSpace Summary data displayed to the user.
5. Guaranteed Post-condition If the user is not able to view the data, a page is displayed describing the error.
