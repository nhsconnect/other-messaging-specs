RequestService
The Request Service message represents a request for a service from a service requester to the Choose and Book Service.

This identifier is globally unique across all records.

It identifies this instance of a service request.

The Identifier Global datatype flavour is used to carry this information as follows:

  • A UUID is held in the root attribute.

  • Fixed value: "PAT"
[1..1] id (II)

Provides NHS Number, the nationally recognised identifier for the patient.

The Identifier External datatype flavour is used to carry this information as follows:

  • The root attribute will contain an OID with the value "2.16.840.1.113883.2.1.4.1"
  • The extension attribute will contain the NHS Number.

Only new, permanent NHS Numbers can be used to identify patients in Choose and Book.
