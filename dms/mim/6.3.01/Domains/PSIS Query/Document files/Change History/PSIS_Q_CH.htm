6    Version 2.0

Previous IM version MIM Version Artefact Id Class Attribute Details of change
1.7 6.1 N/A N/A N/A New HL7 RIM conformant queries and responses.
Query Control Acts now referenced in Infrastructure.
Core PSIS queries modified and new queries added to support functionality as required in spine release 2008A, i.e. extra parameters for date-range and document state, and addition of CRE-type and Synchronisation query.
Queries have been renamed from 'Event' to 'Document' queries to include references to new CDA-style documents.
DPA Subject Access queries renamed to Complete Record Query.
Please reference the PSIS-Query BT LIM located in External Documents.
New format MIF files now included for all messages.
