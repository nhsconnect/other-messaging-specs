Organization.id

Parameter for specifying prescriptions issued, downloaded or dispensed by the identified Prescriber or Dispenser organisation.

Used by: Query
[1..1] value (II)

This is the prescriber site id (NACS code) if the requester organisation is a prescriber or the dispensing site id (NACS code) if the requester organisation is the downloading or the last dispensing site.

The Identifier External datatype flavour is used to carry this information as follows:

  • the root attribute shall contain the value of the NHS SDS Organisation Id OID = 1.2.826.0.1285.0.1.10;
  • the extension attribute shall contain the SDS Organisation Id, known in the External Interface Specification as NACS Organisation Code.
[1..1] semanticsText (ST)

A literal text string identifying the parameter item.

Prescription.id

 Parameter item for selecting the identified prescription.

Used by: Query
[1..1] value (II)

Prescription identifier that is universally unique across all records.  

This shall be the 18 character Short Form Prescription Identifier where it exists.

The Identifier External datatype flavour is used to carry this information as follows:

  • the root attribute shall contain the value of the Short Form Prescription Identifier OID = 2.16.840.1.113883.2.1.3.2.4.18.8
  • the extension attribute shall contain the 18 character Short Form Prescription Identifier

The 36 character clinical statement DCE UUID of the prescription shall only be used where the Short Form Prescription Identifier does not exist.  

The Identifier Global datatype flavour is used to carry this information as follows:

  • the root attribute shall contain the value of the DCE UUID.
[1..1] semanticsText (ST)

A literal text string identifying the parameter item.
