PORX_RM142005UK30
ReimbursmentClaim

This is the message sent from the ETP component to the Reimbursement Authority. It consists of various received messages 'wrapped' together and precursored by a reimbursement claim.

Reference: W5 14.4.7.1

AbstractPSClaim

Abstract placeholder for a PS Claim Information message payload. 

The placeholder and its contents are never instantiated, but are instead replaced by an actual PS Claim Information message from the following set of allowable types and versions:  

  • PORX_MT142002UK05
  • PORX_MT142002UK31

The type and version of the replacement message is indicated by the typeId attribute and must match the value of the templateId of the component link.
