AgentOrgSDS

Identifies an Organization fulfilling a specific role.  Full details of the organization are available from SDS.  In this case, the organisation is actually an SDS Workgroup.

Used by:
[1..1] classCode (CS {CNE:AGNT} ){ Fixed="AGNT " }

Indicates that this is an Agent.

  • Fixed value: "AGNT"
[0..1] code (CV {CNE:RoleCode} )

A code identifying the type of role that the agent is playing.

[1..1] agentWorkgroupSDS (WorkgroupSDS)

A link to details of the Workgroup playing the role.

[1..1] representedOrganizationSDS (OrganizationSDS)

A link to details of the Organisation that scopes the role (i.e. the Organisation that owns the Workgroup)

[1..1] determinerCode (CS {CNE:INSTANCE} ){ Fixed="INSTANCE " }

Indicates that this is an instance of organization entity.

  • Fixed value: "INSTANCE"
[1..1] id (II)

A valid SDS identifier that uniquely identifies the Workgroup.

An SDS Workgroup shall be identified as follows:

  • The root attribute shall contain an OID with the value 1.2.826.0.1285.0.2.0.109
  • the extension attribute shall contain the SDS Workgroup ID.
[1..1] name (ON)

The name of the Workgroup.

OrganizationSDS

Indicates that this is an instance of organization entity.

  • Fixed value: "INSTANCE"
[0..1] id (II)

A valid SDS identifier that uniquely identifies the Organization.

An SDS Organization shall be identified as follows:

  • The root attribute shall contain an OID with the value 1.2.826.0.1285.0.1.10
  • the extension attribute shall contain the SDS Organisation Id, known in the External Interface Specification as the NACS Organisation Code.
[1..1] name (ON)

The name of the Organization. The flavour of ON to be used is:
