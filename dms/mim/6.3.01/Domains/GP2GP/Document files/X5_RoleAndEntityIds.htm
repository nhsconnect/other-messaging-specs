Organization[0..1] OPTIONALOPTIONALOPTIONAL

A national identifier for the organisation (e.g. an OCS identifier). This should be included where available. In this case the root element of the identifier is an OID identifying the identification scheme and the extension element is the national number.

A code identifying the type of organization.

E.g. "NHS Trust"

Note that an organisation may perform different roles. The type of organisation is however constant for a given organisation.

If the code is not present, a description shall be given in the desc element.
