2.7    LinkAssertion - COCD_TP146007UK04

Template used to assert a link between 2 clinical statements

 

               

Note:
Suppliers are not required to implement Link Assertion for systems deployed against Spine Release 2008A. This is because the limited     requirement to code information for that release (only Medications and Allergies & Adverse Drug Reactions) means that the environment does not support a clear Business Use Case for Link Assertion. However, it should be assumed that this will change as the level of coded information increases.
