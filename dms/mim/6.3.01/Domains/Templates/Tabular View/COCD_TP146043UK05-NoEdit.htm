COCD_TP146043UK05
SealingEvent

A record of a sealing event where one or more items of information have been sealed.

NOTE: This is not the sealing event act, but a notification (intended to be sent to PSIS) for recording that a sealing event has occured. Sealing messages are now defined in the EIS.

A fixed code from the SealingActType vocabulary indicating that this is a sealing event.

The Coded Value (CV) with CDA URL datatype/flavour shall be used to carry this information as follows:

  • The code attribute shall contain a code from the relevant vocabulary
  • The codeSystem attribute shall contain the OID for the vocabulary
  • The displayName attribute shall carry the description associated with the code
  • The originalText element can be used to carry a reference element. This can link the coded item to the relevant text in the CDA section
Note the implementation guidance for the domain where this template is used should be consulted regarding correct usage of the originalText element.

Specifies how this author contributes to the context of the request to seal and whether it may be propagated to descendent acts whose association allows such propagation.

"O" (Overriding)- the association replaces any context of the same or more specific type that has been conducted from the parent act.

"P" (Propagating)- the association is conducted to all child acts where the ActRelationship.contextConductionInd is set to "true".
