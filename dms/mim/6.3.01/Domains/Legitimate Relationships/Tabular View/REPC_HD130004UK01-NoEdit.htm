[1..1] code (CV {CNE:SDSRegistrationType} )

A code to provide the type of registration.

In this context, the type of registration is the addition of a user (User Role Profile) to a Workgroup.

The Coded with Code System datatype flavour is used to carry this information as follows:

  • The code attribute will contain value "01" (Add User To Workgroup) from the list of codes in the SDSRegistrationType vocabulary;
  • The codeSystem attribute will contain the OID referenced in the SDSRegistrationType vocabulary.

[1..1] code (CV {CNE:SDSObservationType} )

A code to provide the type of observation.

In this context, the type of observation is the workgroup membership status of the User Role Profile, specifically whether or not they were already a member of the workgroup specified in the original Add User To Workgroup Request.

The Coded with Code System datatype flavour is used to carry this information as follows:

  • The code attribute will contain value "02" (Already a member of the identified workgroup) from the list of codes in the SDSObservationType vocabulary;
  • The codeSystem attribute will contain the OID referenced in the SDSObservationType vocabulary.
[1..1] value (BL)

If present and set to 'true', indicates that the URP is already a member of the workgroup (the workgroup is already present in the identified URP), and that no action was taken. If not present, indicates that the request was successful and the URP was added to the workgroup.
