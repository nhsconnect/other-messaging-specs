[1..1] code (CV {CNE:RoleCode} )

The SDS Job Role Code and Job Role Name.

The Coded Plain flavour of the Coded Value (CV) datatype shall be used to carry this information as follows:

  • The code attribute shall contain the SDS Job Role, known in the External Interface Specification as Job Role Code
  • The codeSystem attribute shall contain an OID with the value 1.2.826.0.1285.0.2.1.104
  • The displayName attribute shall carry the Job Role Name.

Example: <value code="S0030:G0100:R0570" codeSystem="1.2.826.0.1285.0.2.1.104" displayName="'Nursing & MW':'Nurse':'Nurse Consultant'"/>
