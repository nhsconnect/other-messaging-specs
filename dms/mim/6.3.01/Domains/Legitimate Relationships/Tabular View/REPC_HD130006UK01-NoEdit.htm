A code to provide the type of registration.

In this context, the type of registration is the removal of a user (User Role Profile) from a Workgroup.

The Coded with Code System datatype flavour is used to carry this information as follows:

  • The code attribute will contain value "02" (Remove User From Workgroup) from the list of codes in the SDSRegistrationType vocabulary;
  • The codeSystem attribute will contain the OID referenced in the SDSRegistrationType vocabulary.

[1..1] code (CV {CNE:SDSObservationType} )

A code to provide the type of observation.

In this context, the type of observation is the workgroup membership status of the User Role Profile, specifically whether or not they were not actually a member of the workgroup specified in the original Remove User From Workgroup Request.

The Coded with Code System datatype flavour is used to carry this information as follows:

  • The code attribute will contain value "03" (Was not a member of the identified workgroup) from the list of codes in the SDSObservationType vocabulary;
  • The codeSystem attribute will contain the OID referenced in the SDSObservationType vocabulary.
[1..1] value (BL)

If present and set to 'true', indicates that the URP was not a member of the workgroup (the workgroup was not found in the identified URP), and that no action was taken. If not present, indicates that the request was successful and the URP was removed from the workgroup.
