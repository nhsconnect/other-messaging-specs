QUPC_RM010102UK01
Query

Query to the National System to retrieve the User Role Profiles belonging to the specified user, together with the associated Organisation Code and Job Role.

Used by:
[1..1] parameterList (ParameterList)

Link to Parameter Items that are grouped together.

[1..1] userID (UserID)

The SDS User Identifier of the user whose URPs are being requested.

ParameterList

Link to Parameter Items that are grouped together. If neither userRoleProfileID nor organisationCode are present, all of the user's URPs are returned.

Used by: Query
[0..1] organizationCode (OrganizationCode)

If specified, causes only URPs with this Organisation Code to be returned.

[0..1] userRoleProfileID (UserRoleProfileID)

If the identity of the URP is known it can be specified here to define the required URP uniquely. NB If present, Organisation Code cannot be present.

UserID
Used by: Query
