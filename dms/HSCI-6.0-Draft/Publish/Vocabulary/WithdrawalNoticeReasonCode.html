WithdrawalNoticeReasonCode

Code SystemId: 2.16.840.1.113883.2.1.3.2.4.17.522
Version: 1.0
Status: active
Date: 2015-08-04

Description

A set of codes to detail the reason for a withdrawl notice of either an assessment notification or a discharge notification.

Values

Code Description Note
PD The person has died

OA The person wishes to make their own arrangements

CD The person's condition has deteriorated

CHC Eligible for continuing healthcare

CONCHC The person is now being considered for continuing healthcare

NOTLA The person is an ordinary resident in another local authority

CPI Social Services care package inadequate

SELFD The person has discharged themselves

NIE Notification sent in error

NONEED No longer in need of social care services

INTERCARE Transferred to intermediate care

XFRD The person has transferred to another hospital

PWC Withdrawn consent to social services involvement

OFT Other - Free Text
