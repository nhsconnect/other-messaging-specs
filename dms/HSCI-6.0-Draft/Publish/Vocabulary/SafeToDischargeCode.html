SafeToDischargeCode

Code SystemId: 2.16.840.1.113883.2.1.3.2.4.17.271
Version: 2.0
Status: active
Date: 2015-10-07

Description

A set of codes to determine if the patient is safe to discharge.

Values

Code Description Note
STD The patient is considered safe to discharge or safe to transfer

STDNFC The patient has been discharged and no further care planning is required

Retired from Publication as from 30-Oct-2015
STDFC The patient has been discharged and further care planning is still needed

Retired from Publication as from 30-Oct-2015
