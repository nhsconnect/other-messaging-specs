NHSContinuingHealthcareAcceptResponse Tabular View

COMT_RM000010GB02
CHCAcceptResponse

A message used to accept an NHS Continuing Healthcare CDA Document.
Used by:
[1..1] [M] classCode (CS { CNE: ActClass } )
{ Fixed="INFRM" }

Indicates this is information.
[1..1] [M] moodCode (CS { CNE: ActMood } )
{ Fixed="EVN" }

Indicates this is an event.
[1..1] [M] code (CV { CNE:CHCAcceptMessageTypeSnCT } )

A code from the vocabulary shown indicating that this is an NHS Continuing Healthcare Accept Response.

A SnomedCT code taken from the subset shown.

The Coded with Code System flavour of Coded Value (CV) datatype shall be used to carry this information as follows:

  • The code attribute shall contain the value.
  • The codeSystem attribute shall contain the OID for the SnomedCT concerned.
[1..1] [M] effectiveTime (TS)

The time the acceptance response was sent.

In the format yyyymmddhhmmss
[1..1] [M] id (II)

The id shall be the identifier of this NHS Continuing Healthcare Accept Response.

  • The root attribute shall contain a DCE UUID.
[1..1] [M] author

The author of the NHS Continuing Healthcare Assessment Document Accept Response.
[1..1] [M] typeCode (CS { CNE: ParticipationType } )
{ Fixed="AUT" }
Indicates this is the Author.
[1..1] [M] contextControlCode (CS { CNE: ContextControl } )
{ Fixed="OP" }

Specifies how the author recipient contributes to the context of NHS Continuing Healthcare Accept Response and whether it may be propagated to descendent acts whose association allows such propagation.

"O" (Overriding)- the association replaces any context of the same or more specific type that has been conducted from the parent act.

"P" (Propagating)- the association is conducted to all child acts where the ActRelationship.contextConductionInd is set to "true."
