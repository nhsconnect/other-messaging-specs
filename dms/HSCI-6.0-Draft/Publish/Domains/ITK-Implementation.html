Trigger Events

Assessment Notice (COMT_TE010020UK02)

The Assessment Notice trigger event indicates that a Hospital Assessment Notice document and related data is ready to send from the Hospital System to Social Care.

Assessment Notice Accept Response (COMT_TE010021UK02)

The Assessment Notice Accept Response trigger event indicates that a Hospital Assessment Notice Accept Response message is ready to send from the Social Care System.

Assessment Notice Reject Response Under (COMT_TE010022UK02)

The Assessment Notice Reject Response trigger event indicates that a Hospital Assessment Notice Rejection Response message is ready to send from the Social Care System.

Discharge Notice (COMT_TE010023UK02)

The Discharge Notice trigger event indicates that a Hospital Discharge Notice document and related data is ready to send from the Hospital System to Social Care.

Discharge Notice Agreement (COMT_TE010024UK02)

The Discharge Notice Agreement trigger event indicates that a Hospital Discharge Notice Agreement message is ready to send from the Social Care System.

Discharge Notice Disagreement (COMT_TE010025UK02)

The Discharge Notice trigger event indicates that a Hospital Discharge Notice Disagreement message is ready to send from the Social Care System.

Withdrawal Notice (Discharge Notice Only) (COMT_TE010026UK02)

The Withdrawal Notice (Discharge Notice Only) trigger event indicates that a Withdrawal Notice for Discharge Notice message is ready to send from the Hospital System to Social Care.

Withdrawal Notice (Complete Process) (COMT_TE010027UK02)

The Withdrawal Notice (Complete Process) trigger event indicates that a Withdrawal Notice for the complete process message is ready to send from the Hospital System to Social Care.

Withdrawal Notice Accept Response (COMT_TE010030GB01)

The Withdrawal Notice Accept Response trigger event indicates that a Withdrawal Notice Accept Response message is ready to send from the Social Care System, to convey that the Withdrawal Notice has been accepted.

Withdrawal Notice Reject Response (COMT_TE010031GB01)

The Withdrawal Notice Reject Response trigger event indicates that a Withdrawal Notice Reject Response message is ready to send from the Social Care System, to convey that the Withdrawal Notice has been rejected.

Interactions

Care Act 2014 Notices to Social Care Interactions

This component covers message specifications for Assessment and Discharge Notices under the Care Act 2014 and the associated communications between health care and social care.


Assessment Notice Interactions

Note : Message acknowledgements and responses are requested by the document sender via handlingSpecifications within the ITK Distribution Envelope. For more detail on handlingSpecifications and the acknowledgement framework see the ITK Core pack (available from TRUD).

Discharge Notice (urn:nhs-itk:interaction:DischargeNotice-v3-0)  

Communicates a Discharge Notice to a recipient Social Care System.

On Receipt of this interaction the recipient must send an Business Acknowledgment (Version 1.0) if requested by the sender.

AND

Send either a Discharge Notice Agreement (Version 3.0)

OR

Send a Discharge Notice Disgreement (Version 3.0)

Sending of message.


Sender: Hospital System (COMT_AR010020UK01)
Receiver: Social Care System (COMT_AR010021UK01)
Message:
  • Wire Format: (COMT_MT000008UK02)
  • Conformance Format: (COMT_MT000008UK02)
