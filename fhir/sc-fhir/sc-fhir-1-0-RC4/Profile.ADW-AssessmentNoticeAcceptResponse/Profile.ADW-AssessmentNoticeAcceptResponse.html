Version 1.0 : RC 4

    The following structure definitions are available in this profile.

    NameTypeDescription & Constraints
    ADW-Message-Bundle-1-0BundleA Bundle resource constrained to meet the requirements of ADW messaging.
    ADW-Assessment-Notice-Accept-Response-Message-Header-1-0MessageHeaderThe Assessment Notice Accept Response Profile is a constrained version of the MessageHeader resource and carries the Assessment Notice Accept Response MessageHeader information.
    ADW-Referral-Request-Accepted-1-0ReferralRequestThe ADW Referral Request Profile is a constrained version of the Referral Request resource and carries information about the referral request that has been accepted.
    ADW-Patient-1-0PatientThe ADW Patient Profile is a constrained version of the Patient resource and carries details of the patient who is the subject of the ADW communication.
    ADW-Practitioner-1-0PractitionerThe ADW Practitioner Profile is a constrained version of the Practitioner Resource to provide a standard for an ADW Hospital Liaison or Social Worker practitioner.
    ADW-Organization-1-0OrganizationThe Organization Profile is a constrained version of the Organization Resource to provide a standard for an ADW hospital or Local Authority organization.
    ADW-Social-Services-Team-Organization-1-0OrganizationThe Organization Profile is a constrained version of the Organization Resource to provide a standard for an ADW Social Services team.
    ADW-Response-Encounter-1-0EncounterThe Response Encounter profile is a constrained version of the Encounter resource which shows the links between the patient and the Hospital Ward and Hospital Service provider in the context of an ADW message.
    ADW-Location-1-0LocationThe Location Profile is a constrained version of the Location Resource to provide a standard for a hospital ward within the context of an ADW message.
