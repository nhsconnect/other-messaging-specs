    GPSoC-AppointmentResponse-1-0

    The Appointment Response resource provides information to a reply for an appointment request.

    Follow this link to view examples for AppointmentResponse: Examples
    NameCard.TypeDescription & Constraints

    Fixed Value: http://fhir.nhs.net/Id/gpsoc-appointment-response-identifier

    Example Value: PY6878

    display0..1stringThe Text alternative for the resource

    end1..1instantDate/Time that the appointment is to conclude, or requested new end time
