Version 1.0 : DRAFT B

    GPSoC-Location-1-0

    The Location resource provides information and details on the physical location and the services required for an appointment.

    Follow this link to view examples for Location: Examples
    NameCard.TypeDescription & Constraints
    Location1..1DomainResourceThis location resource is constrained to support communication of appointment and scheduling details with in scope of the GPSoC IM2 project.

    identifierUnique code or number identifying the location to its users
    Slice: Open at End

    identifier (ODS Site Code)0..1IdentifierODS Site code to identify the organisation at site level.
