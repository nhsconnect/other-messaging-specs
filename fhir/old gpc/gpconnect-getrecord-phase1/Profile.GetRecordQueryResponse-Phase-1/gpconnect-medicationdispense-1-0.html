Version 1.0 : DRAFT B

    GPConnect-MedicationDispense-1-0

    This Medication Dispense resource is used to describe a medication product that is/ has been dispensed for a patient and the instructions on how it should be administered. The medication dispense is the result of a pharmacy system responding to a medication order.
