    GPConnect-Patient-Identifier-Type-1-0

    Default Scheme: GP Connect Patient Identifier Type

    System URI: http://fhir.nhs.net/ValueSet/gpconnect-patient-identifier-type-1-0

    Description

    A vocabulary of the other identifiers which can be used to identify a Patient.

    Code Description
    AMS Army Medical Services Number
    CHI Community Health Index Number
    HC Health and Care Number
    Local Local Identifier
    MOD SN Ministry of Defence Service Number
    ONHS Old Format NHS Number
    PN Prison Number
    RAFN Royal Air Force Number
    RN Royal Navy Number
    SSD Social Security Disability Number
    UPN Unique Pupil Number
