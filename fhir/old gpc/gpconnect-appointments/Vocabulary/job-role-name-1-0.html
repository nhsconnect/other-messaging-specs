    Job-Role-Name-1-0

    Default Scheme: Job Role Name

    Assigned OID: 2.16.840.1.113883.2.1.3.2.4.17.124

    System URI: http://fhir.nhs.net/ValueSet/job-role-name-1-0

    Description

    Identifies the job role associated with the person where the information is not known to SDS.

    This may contain any code which describes the person's job role
