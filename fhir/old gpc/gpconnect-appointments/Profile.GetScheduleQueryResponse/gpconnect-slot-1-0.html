Version 1.0 : DRAFT B

    GPConnect-Slot-1-0

    The Slot resource provides time-slots on a schedule that may be available to book for an appointment.

    Follow this link to view examples for GPConnect-Slot-1-0: Examples
    NameCard.TypeDescription & Constraints
    SlotDomainResourceThe Slot resource provides time-slots on a schedule that may be available to book for an appointment

    identifier (Slot Identifer)0..1IdentifierAn identifier used to identify the slot

    system0..1UriThe namespace for the identifier
    Example Value: <identifier>

    value1..1StringThe value that is unique
    Example Value: Slot001

    type0..1CodeableConceptThe type of appointment that is being booked.
    Binding: Additional details about where the content was created (e.g. clinical specialty) (http://hl7.org/fhir/ValueSet/c80-practice-codes)
    Binding Strength: Example

    text0..1StringPlain text representation of the concept

    schedule1..1gpconnect-schedule-1-0The schedule resource that this slot defines an interval of status information

    reference1..1StringRelative URL reference
    Example Value: Schedule/0A228540-99AB-11E5-A837-0800200C9A66

    display0..1StringText alternative for the resource

    freeBusyType1..1CodeBUSY | FREE | BUSY-UNAVAILABLE | BUSY-TENTATIVE
    Binding: The free/busy status of a slot (http://hl7.org/fhir/ValueSet/slotstatus)
    Binding Strength: Required
    Example Value: free

    start1..1InstantDate/Time that the slot is to begin
    Example Value: 0001-01-01T00:00:00+00:00

    end1..1InstantDate/Time that the slot is to conclude

    overbooked0..1BooleanThis slot has already been overbooked, appointments are unlikely to be accepted for this time

    comment0..1StringComments on the slot to describe any extended information. Such as custom constraints on the slot
    Example Value: This slot is only to be used for physiotherapy appointments.
