Version 1.0 : DRAFT A

    GPConnect-OperationOutcome-1-0

    The Operation Outcome resource supports a collection of error, warning or information messages that result from a GP Connect Query Request.

    Follow this link to view examples for GPConnect-OperationOutcome-1-0: Examples
    NameCard.TypeDescription & Constraints
    OperationOutcomeDomainResourceInformation about the success/failure of an action

    issue1..*BackboneElementA single issue associated with the action

    severity1..1Codefatal | error | warning | information
    Binding: How the issue affects the success of the action. (http://hl7.org/fhir/DSTU2/valueset-issue-severity.html)
    Binding Strength: Required

    code1..1CodeableConceptError or warning code
    Binding: A code that describes the type of issue. (http://hl7.org/fhir/DSTU2/valueset-issue-type.html)
    Binding Strength: Required

    details1..1CodeableConceptGP Connect Error or Warning code
    Binding: A coded expression of the type of issue (http://fhir.nhs.net/ValueSet/gpconnect-response-code-1-0)
    Binding Strength: Required

    coding1..1CodingCode defined by a terminology system

    system1..1UriGP Connect Response Codes
    Fixed Value: http://fhir.nhs.net/ValueSet/gpconnect-response-code-1-0

    code1..1CodeSymbol in syntax defined by the system

    display1..1StringRepresentation defined by the system

    text0..1StringPlain text representation of the GP Connect Error or Warning code

    diagnostics0..1StringAdditional diagnostic information about the issue
