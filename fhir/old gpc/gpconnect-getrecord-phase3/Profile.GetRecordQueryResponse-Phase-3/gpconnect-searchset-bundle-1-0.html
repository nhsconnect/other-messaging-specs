    GPConnect-Searchset-Bundle-1-0

    The 'GPConnect-Searchset-Bundle-1-0' Bundle resource profile is used in the GP Connect outbound interface as a container to collect a combination of the GP Connect Query Response resources, the combination and order of which is dependent on the GP Connect Query Response interaction:
