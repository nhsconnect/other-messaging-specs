Version 1.0 : DRAFT D

    The following structure definitions are available in this profile.

    NameTypeDescription & Constraints
    GPConnect-Searchset-Bundle-1-0BundleThis Bundle resource is constrained to represent a container to collect a combination of the GP Connect Query Response resources.
    GPConnect-OperationOutcome-1-0OperationOutcomeThe Operation Outcome resource supports a collection of error, warning or information messages that result from a GP Connect Query Request.
    GPConnect-Patient-1-0PatientThe Patient resource carries the patient demographic details.
    GPConnect-CareRecord-Composition-1-0CompositionThe Composition resource carries information in the patient's record.
    GPConnect-AllergyIntolerance-1-0AllergyIntoleranceThe Allergy Intolerance resource is used to record detailed information about allergies and adverse reactions recognized by a clinician.
    GPConnect-Diagnosis-Condition-1-0ConditionThe Diagnosis Condition resource is used to record detailed information about diagnoses recognized by a clinician.
    GPConnect-Encounter-1-0EncounterThe Encounter resource is used to record the interaction between the patient and a healthcare professional.
    GPConnect-Immunization-1-0ImmunizationThe Immunization resource is used for tracking the current and historical vaccinations that have been administered to a patient.
    GPConnect-Investigation-DiagnosticOrder-1-0DiagnosticOrderThe Diagnostic Order resource is used to record a request for a diagnostic investigation to be performed on a patient.
    GPConnect-KeyIndicator-Flag-1-0FlagThe Flag resource is used to record the risks and flags for a patient.
    GPConnect-Medication-1-0MedicationThe Medication resource is used to identify and define the medication for a patient.
    GPConnect-MedicationOrder-1-0MedicationOrderThe Medication Order resource is used to capture the order and instructions for administration of prescribed medications to a patient.
    GPConnect-MedicationDispense-1-0MedicationDispenseThe Medication Dispense resource is used to describe a medication product that is/has been dispensed for a patient and the instructions on how it should be administered. The Medication Dispense is the result of a pharmacy system responding to a Medication Order.
    GPConnect-MedicationAdministration-1-0MedicationAdministrationThe Medication Administration resource is used for tracking the medicine a patient consumes or medicine that has been administered to them.
    GPConnect-Observation-1-0ObservationThe Observation resource is used for tracking the current and historical observations that have been made for a patient.
    GPConnect-Problem-Condition-1-0ConditionThe Problem Condition resource is used to record detailed information about problems recognized by a clinician.
    GPConnect-Procedure-1-0ProcedureThe Procedure resource is used for tracking the current and historical procedures that have been performed on a patient.
    GPConnect-Practitioner-1-0PractitionerThe Practitioner resource represents the healthcare professional directly or indirectly involved in the provision of healthcare related services.
    GPConnect-Location-1-0LocationThe Location resource provides information and details on the physical location and the services provided.
    GPConnect-Organization-1-0OrganizationThe Organization resource represents the organisation that employs the healthcare professional.

    The following extensions are used within these resource constraints.

    NameTypeDescription & Constraints
    GPConnect-Allergy-End-Date-1-0ExtensionThis Extension resource is constrained to support the date that the allergy was no longer valid.
    GPConnect-Encounter-Reference-1-0ExtensionThis Extension resource is constrained to support an encounter reference for the Allergy Intolerance resource.
    GPConnect-Flag-Information-Location-1-0ExtensionThis Extension resource is constrained to support the location where further information about the flag can be found.
    GPConnect-Flag-Source-1-0ExtensionThis Extension resource is constrained to support the source of the information on the flag.
    GPConnect-Medication-Quantity-Text-1-0ExtensionThis Extension resource is constrained to support the medication quantity in text form, for the GP Connect API interaction requirements.
    GPConnect-Prescription-Last-Issue-Date-1-0ExtensionThis Extension resource is constrained to support the date a prescription was last issued.
    GPConnect-Prescription-Repeat-Review-Date-1-0ExtensionThis Extension resource is constrained to support the date a repeat prescription will be reviewed.
