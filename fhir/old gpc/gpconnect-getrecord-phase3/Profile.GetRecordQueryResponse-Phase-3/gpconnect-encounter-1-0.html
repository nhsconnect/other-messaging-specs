    GPConnect-Encounter-1-0

    The Encounter resource is used to record the interaction between the patient and a healthcare professional.

    Follow this link to view examples for GPConnect-Encounter-1-0: Examples
    NameCard.TypeDescription & Constraints
    EncounterEncounterAn interaction during which services are provided to the patient

    identifier1..1IdentifierA unique identifier for the encounter

    system0..1UriThe namespace for the identifier
    Example Value: http://www.acme.com/identifiers/patient or urn:ietf:rfc:3986 if the Identifier.value itself is a full uri

    value0..1StringThe value that is unique
    Example Value: 123456
