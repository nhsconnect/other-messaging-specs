Version 1.0 : DRAFT D

    GPConnect-MedicationDispense-1-0

    The Medication Dispense resource is used to describe a medication product that is/has been dispensed for a patient and the instructions on how it should be administered. The Medication Dispense is the result of a pharmacy system responding to a Medication Order.
