    GPConnect-CareRecord-Operation-1-0

    This Operation resource carries information on the Get Record Query request parameters.

    Follow this link to view examples for GPConnect-CareRecord-Operation-1-0: Examples

    Meta

    Name Type Value
    Name String GPConnect-CareRecord-Operation-1-0
    Kind Operation Operation
    Description String Request to retrieve the Patient's Care Record
    Code Code getcarerecord
    System Boolean False
    Instance Boolean False

    Input Parameters

    Name Card. Type Description
    patientNHSNumber 1..1 Identifier Patient that matches the NHS Number
    recordSection 0..* string Section of the care record defined by the parameter
    timePeriod 0..1 Period The time period for the requested information

    Output Parameters

    Name Card. Type Description
    response 1..1 Bundle View of the Patient's care record. This can be either a view of the patient's care record or specified sections of the care record.
