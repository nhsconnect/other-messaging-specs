    CareConnect-GPC-Condition-1

    This Condition resource has been constrained to represent the CareConnect Problem Condition profile, with further constraints applied to support the GP Connect requirements.

    View Condition Resource Changes for a delta of changes made to the standard FHIR DSTU2 Condition resource for use within the GP Connect APIs.
