    CareConnect-GPC-Immunization-1

    This Immunization resource has been constrained to represent the CareConnect Immunization profile, with further constraints applied to support the GP Connect requirements.

    View Immunization Resource Changes for a delta of changes made to the standard FHIR DSTU2 Immunization resource for use within the GP Connect APIs.
