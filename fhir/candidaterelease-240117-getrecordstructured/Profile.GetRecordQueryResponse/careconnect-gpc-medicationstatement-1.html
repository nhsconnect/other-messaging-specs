    CareConnect-GPC-MedicationStatement-1

    This MedicationStatement resource has been constrained to represent the CareConnect MedicationStatement profile, with further constraints applied to support the GP Connect requirements.

    View MedicationStatement Resource Changes for a delta of changes made to the standard FHIR DSTU2 MedicationStatement resource for use within the GP Connect APIs.
