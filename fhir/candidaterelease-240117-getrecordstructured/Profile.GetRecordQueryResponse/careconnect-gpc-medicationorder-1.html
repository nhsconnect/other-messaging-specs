    CareConnect-GPC-MedicationOrder-1

    This MedicationOrder resource has been constrained to represent the CareConnect MedicationOrder profile, with further constraints applied to support the GP Connect requirements.

    View MedicationOrder Resource Changes for a delta of changes made to the standard FHIR DSTU2 MedicationOrder resource for use within the GP Connect APIs.
