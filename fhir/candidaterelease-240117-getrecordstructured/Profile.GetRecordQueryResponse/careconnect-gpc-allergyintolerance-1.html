    CareConnect-GPC-AllergyIntolerance-1

    This AllergyIntolerance resource has been constrained to represent the CareConnect AllergyIntolerance profile, with further constraints applied to support the GP Connect requirements.

    View AllergyIntolerance Resource Changes for a delta of changes made to the standard FHIR DSTU2 AllergyIntolerance resource for use within the GP Connect APIs.

    Name Type Description & Constraints
    encounter-associatedEncounter Extension This Extension resource is constrained to support an encounter reference for the Allergy Intolerance resource
    Extension-CareConnect-AllergyIntoleranceEndDate-1 Extension This Extension resource is constrained to support the date that the allergy/intolerance was no longer valid.
    Extension-CareConnect-Evidence-1 Extension This Extension resource is constrained to support a reference to results of investigations that confirmed the certainty of the diagnosis. Examples might include results of skin prick allergy tests
    Extension-CareConnect-ProbabilityOfRecurrence-1 Extension This Extension resource is constrained to support representation of the probability of the reaction (allergic, adverse, intolerant) recurring.
    Extension-Coding-SCTDescID Extension This Extension resource is constrained to support SNOMED CT Description ID for the display
    Extension-CareConnect-AllergyCertainty-1 Extension This Extension resource is constrained to support a coded certainty of the allergy/intolerance.
    Extension-CareConnect-AllergySeverity-1 Extension This Extension resource is constrained to support a coded severity of the allergy/intolerance.
    Extension-Coding-SCTDescID Extension This Extension resource is constrained to support SNOMED CT Description ID for the display
