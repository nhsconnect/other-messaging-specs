Ethnicity

Code SystemId: 2.16.840.1.113883.2.1.3.2.4.16.81
Version: 1.0
Status: active
Date: 2014-01-15

Description

Default Scheme : NHS Data Model and Dictionary for England, ETHNIC CATEGORY CODE

The ethnicity of a person, as specified by the person, for values defined by the NHS Data Model and Dictionary for England. This vocabulary is based on the classification used for the 2001 census, replacing ETHNIC GROUP in the Commissioning Data Set Flows.

Note: Where there is a requirement to use an extended set of values then the EthnicCategory vocabulary maybe used.
