EntityDeterminer

Code SystemId:
Version: 1.0
Status: active
Date: 2011-12-21

Description

This vocabulary contains a set of codes to distinguish whether any given Entity object represents a type of object or specific one.

Values

Code Description Note
INSTANCE specific

An Entity which has a specific (instance) determiner represents one specific instance of a thing .

Examples, St Elsewhere Trust Surgical Outpatient Clinic, a specific ECG machine (e.g. identified by serial number), Mr Sherlock Holmes.
KIND described

An Entity which has a described (kind) determiner represents a general description of a kind of thing. It may refer to a whole or part of such a kind of thing, or to multiple occurrences of this kind of thing.

Examples, a surgical clinic, an ECG machine, a person, a group of people.
QUANTIFIED_KIND described quantified

An Entity which has a described quantified (quantified kind) determiner represents a general description of a specific amount of a thing.

Examples, of syringe (quantity = 3,) stands for exactly three syringes.
