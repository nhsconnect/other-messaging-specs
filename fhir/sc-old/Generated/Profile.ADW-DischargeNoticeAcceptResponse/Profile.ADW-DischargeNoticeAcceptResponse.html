1.0 : DRAFT C

    The following structure definitions are available in this message profile.

    NameTypeDescription & Constraints
    ADW-Message-Bundle-1-0BundleA Bundle resource constrained to meet the requirements of ADW messaging.
    ADW-DischargeNoticeAcceptResponse-Message-Header-1-0MessageHeaderThe Discharge Notice Accept Response Profile is a constrained version of the Message Header resource and carries the Discharge Notice Accept Response Message header information
    ADW-ReferralRequest-Accepted-1-0ReferralRequestThe ADW Referral Request Profile is a constrained version of the ReferralRequest resource and carries the ADW referral request information.
    ADW-Patient-1-0PatientThis patient resource is constrained to provide information about the patient who is the subject of the ADW process
    ADW-Practitioner-1-0PractitionerThe ADW Practitioner Profile is a constrained version of the Practitioner Resource to provide a standard for an Assessment, Discharge and Withdrawl health care professional.
    ADW-Organization-1-0OrganizationThis Organization resource is constrained to represent a healthcare organisation.
