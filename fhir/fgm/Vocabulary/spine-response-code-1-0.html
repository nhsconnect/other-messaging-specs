    Spine-Response-Code-1-0

    System URI: http://fhir.nhs.net/ValueSet/spine-response-code-1-0

    A set of codes to indicate low level error information about a Spine 2 error response to a request for patient record details.

    Code Description
    FGM-0001 No FGM Record Found
    FGM-0002 NHS Number Invalid
    FGM-9999 Message not well formed
