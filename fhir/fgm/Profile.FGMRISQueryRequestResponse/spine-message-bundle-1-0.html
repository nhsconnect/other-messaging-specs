Version 1.0 : RC1

    Spine-Message-Bundle-1-0

    The 'Spine-Message-Bundle-1-0' Bundle resource profile is used in the FGM outbound interface as a container to collect a combination of the FGM RIS Query Response resources, the combination and order of which is dependent on the FGM RIS Query Response interaction:

    If the FGM query response conveys that an FGM entry was found, the 'Spine-Message-Bundle-1-0' bundle contains the FGM RIS Query Response resources in the following combined order:

    • Spine-Response-MessageHeader-1-0 spine-response-messageheader-1-0
    • Spine-RIS-Flag-1-0 spine-ris-flag-1-0
    • Spine-RIS-Patient-1-0 spine-ris-patient-1-0

    If the FGM query response conveys that an FGM entry was not found or that the query was invalid, the 'Spine-Message-Bundle-1-0' bundle contains the FGM RIS Query Response resources in the following combined order:

    • Spine-Response-MessageHeader-1-0 spine-response-messageheader-1-0
    • Spine-OperationOutcome-1-0 spine-operationoutcome-1-0
