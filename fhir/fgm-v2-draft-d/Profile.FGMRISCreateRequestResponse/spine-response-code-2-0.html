    Spine-Response-Code-2-0

    Summary

    Name Description
    Defining URL: http://fhir.nhs.net/ValueSet/spine-response-code-2-0
    Name: Spine-Response-Code-2-0
    Definition: A set of codes to indicate low level error information about a Spine 2 /SMSP error response to a request for patient record details.
    Status: Active
    Last Updated: 20/09/2016 00:00:00 +01:00
    Source Resource:

    XML / JSON


    This valueset includes all the codes for Spine-Response-Code-2-0 ( http://fhir.nhs.net/ValueSet/spine-response-code-2-0 )
